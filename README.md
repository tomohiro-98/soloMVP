# soloMVP
Create React Appを使った始め方
このプロジェクトはCreate React Appで作成されました。

使用可能なスクリプト
プロジェクトディレクトリ内で、以下のコマンドを実行できます:

npm start
開発モードでアプリを実行します。
ブラウザで http://localhost:3000 を開いて表示します。

ファイルを変更するとページがリロードされます。
また、コンソールにはLintエラーが表示される場合があります。

npm test
テストランナーをインタラクティブなウォッチモードで起動します。
詳細は テストの実行 のセクションを参照してください。

npm run build
アプリを本番向けに build フォルダにビルドします。
Reactを本番モードで適切にバンドルし、最適なパフォーマンスを実現します。

ビルドは最小化され、ファイル名にハッシュが含まれます。
これでアプリはデプロイの準備ができました！

詳細は デプロイ のセクションを参照してください。

npm run eject
注意: これは一方向の操作です。eject を実行すると元に戻すことはできません！

もしビルドツールや設定の選択に満足していない場合、いつでも eject を実行できます。このコマンドはプロジェクトからビルドに関する依存関係を削除します。

その代わりに、すべての設定ファイルと依存関係（webpack、Babel、ESLintなど）をプロジェクト内にコピーします。これにより、それらに完全に制御することができます。ただし、eject を除くすべてのコマンドは引き続き動作しますが、それらはコピーされたスクリプトを指すようになるため、必要に応じて調整することができます。この時点で、自己責任になります。

eject を使用する必要はありません。管理された機能セットは、小規模および中規模のデプロイに適しており、準備ができたらカスタマイズする必要はありません。
